Fill gaps where collaged pieces meet, maintaining paper grain direction. Adjust sampling to match fiber patterns.

Photoshop Technique:
  1. Place collage elements on separate layers
  2. Create new layer between them
  3. Select gap with Lasso tool
  4. Use Edit â†’ Content-Aware Fill
  5. Select areas with matching grain direction
  6. Enable "Mirror" for symmetrical textures
  7. Set Structure to 15-25 to maintain grain patterns
  8. Use Default/Low Color Adaptation for paper fibers
Pro Tip: For mixed-media collages, create separate CAF layers for each material type, then blend with masks.

Combine CAF with soft Eraser or Layer Mask to create subtle transitions while preserving handmade quality.

Photoshop Technique:
  1. Position elements on separate layers
  2. Create layer mask on top layer
  3. Paint black with soft brush (30-50% hardness) for gradual fade
  4. Select gap area with Lasso tool
  5. Create new layer and use Edit â†’ Content-Aware Fill
  6. Sample from both elements you're blending
  7. Output to new layer for better control
  8. Use layer mask to refine transition
Alternative: Use Mixer Brush with "Sample All Layers" to manually blend after applying CAF.
